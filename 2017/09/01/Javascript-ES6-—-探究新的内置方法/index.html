Sep 01, 2017点击

翻译于众成翻译

ES6标准发布的新特性中，有大量新的内置方法。这些新方法旨在简化和规范开发人员在使用JS数据类型：Numbers， Strings， Object， Arrays时遇到的一些常见情况。

ES6的一个目标就是使JS编程更简洁更干净。让我们来看一下，ES6是如何帮助我们达到这一目标的。
