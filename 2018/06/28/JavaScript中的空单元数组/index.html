空单元数组：数组中包含一个或者多个空单元的数组（空单元 != undefined）

什么样的操作会导致生成空数组呢？
  • new Array(10)
    像上面一样，调用Array构造函数，然后传入一个数字，这个时候产生的数组就是一个空单元数组。
  • delete arr[2]
    使用delete删除数组的某个元素，并不会改变这个数组的length，响应地，也就产生了空单元数组。
  • arr.length = X
    如果设定某个数组的length属性为某个大于当前元素个数的值，那么该数组也会成为空单元数组。
空单元数组有什么危害？

空单元数组在我们使用数组的forEach、map方法时，可能会产生出人意料的结果。尤其是像上面第二种和第三种情况，因为这个时候，你往往会忽略，你的操作导致了空单元数组的产生。
