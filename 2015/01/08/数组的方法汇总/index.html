数组的方法汇总

Created at 2015-01-08

##数组的方法：

  1. slice()
    • 这个方法用来复制数组中的某些元素，第一个参数为起始的索引，第二个参数为结束的索引，但是要注意，复制出来的这些元素中，是不包括end的。返回的是一个数组，里面包含了复制的那些元素。
    • 这个方法的一个妙用是：[].slice.call(auguments)这样就可以实现将函数实参的这个伪数组转换成真正的数组
    • 针对上面这个需求，还可以使用下面将会提到的Array.from(伪数组)
  2. splice()
    • 这个方法是用来删除数组的元素的，第一个参数是开始删除元素的索引，第二个参数是要删除的元素的长度或者说是个数。

  3. shift()

    • 这个方法是用来删除数组中第一个元素的

  4. unshift()

    • 这个方法是用来在数组的首位添加一个元素的

  5. pop()

    • 这个方法是删除数组的最后一个元素

  6. push()

    • 这个方法是在数组的最后添加用一个元素

  7. join()

    • 这个方法是将数组中的元素用某个标识拼接起来

  8. concat()

    • 这个方法是将另外一个数组中的元素添加到第一个参数数组中

  9. indexOf()

    • 这个方法不只是数组的，字符串中 也可以使用，它是用来查找数组中有没有某个元素的

  10. foreach()

    • 这个是ES6中新增的方法，使用来遍历数组的

  11. map()

    • 这个同样也是用来遍历数组的，而且这个方法可以返回一个经过操作之后的新数组

  12. filter()

    • 过滤，这个方法遍历数组，然后返回我们指定条件的数组元素。当然，返回的依然是一个数组

  13. find()

    • 这个方法用来给出符合条件的第一个元素

  14. findIndex()

    • 这个也是用来返回符合条件的第一个，但是这个方法的返回值是索引

  15. Array.from(伪数组)，这里必须是Array来调用这个方法

    • 这个就是用来将一个伪数组转换成真正的数组

  16. reduce()

    • 这个方法是用来将数组的元素累积。比如var total = [0, 1, 2, 3].reduce(function(a, b) { return a + b; }，0); // total == 6

  17. includes()

    • 这个方法是用来判断数组中是否含有某个元素
