JS中易错点笔记

数组的reverse方法

var arr = [1,2,3];
var arr2 = arr.reverse();
console.log(arr); // [3,2,1]
console.log(arr2); // [3,2,1];

如上面代码所见，数组的这个reverse方法，会反转我们的数组，这没有
什么问题，也没有什么要我们多加注意的地方，但是我们要注意到的一点
是这个方法会将我们的原数组改变，所以我们在以后要用的时候，一定
要注意这一点。

window.close

这个BOM方法可以将我们当前窗口关闭，这在我们用js代码来写，用户是否
确认关闭当前窗口的时候可以用得到。比方说下面的代码：

if (confirm('您确定关闭当前窗口吗？')) {
    window.close();
}

document.forms

这个API是用来获取到页面中的所有的form表单

document.forms[0].elements

这个API使用来获取到页面中第一个表单元素的所有input（textarea）

在写一个正则表达式的时候，用构造函数形式和字面量形式有什么区别？

构造函数模式要比字面量形式复杂，所以还是喜欢用字面量形式。比如说

var req = /\\/  // 表示一个/
var req = new ReqExp('\\\\')  // 表示一个/

ajax请求的状态码

200---服务器成功返回数据
304---页面未修改
404---请求的网页不存在
403---服务器禁止访问
503---服务器暂时不可用
500---服务器内部错误
