原生JS中三种绑定事件的方式

Feb 02, 2016点击

  • btn.onclick = function(){} 所有浏览器都支持
  • addEventListener（事件名称，事件处理程序，useCapture），第三个参数，如果是true，表示捕获，如果是false，表示冒泡，一般用冒泡 IE9以后才支持
  • 事件处理的三个阶段：1. 捕获 2. 目标阶段（执行函数） 3. 冒泡
  • IE中：attachEvent（“on”+事件名称，事件处理程序），这个方法不支持捕获， IE6-10支持
    ####不同的绑定事件的方式，事件处理程序中的this的指向
  • onclick这种方式绑定事件中的this指的是事件源
  • addEventListener这种方式绑定的事件中的this指的也是事件源
  • IE中：attachEvent这种方式绑定的事件中的this的指向是window
    ####绑定事件的兼容处理 
    if (window.addEventListener) {
        target.addEventListener(type,handler)
    } else if (window.attachEvent) {
        target.attachEvent("on"+type, function () {
            handler.apply(target)
        })
    } else {
        //点语法可以用中括号语法来替代
        target["on"+type] = handler;
    }
}

var addEvent = function () {
if (window.addEventListener) {
return function (target,type,handler) {
target.addEventListener(type,handler)
}
} else if (window.attachEvent) {
return function (target,type,handler) {
target.attachEvent(“on”+type, function () {
handler.apply(target)
})
}
} else {
return function (target,type,handler) {
//点语法可以用中括号语法来替代
target[“on”+type] = handler;
}
}
}();
