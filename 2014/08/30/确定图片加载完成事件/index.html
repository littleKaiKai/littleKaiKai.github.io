图片加载完成的两种实现

Aug 30, 2014点击

如果需要等到某个图片加载完成之后，然后再去做一些事情的话：

  1. 可以使用onload事件，用法跟click这样事件的用法是一样的
  2. 也可以使用 
 var img = new Image(); //创建一个Image对象，实现图片的预下载
 img.src = url;
 if(img.complete) { // 如果图片已经存在于浏览器缓存，直接调用回调函数
     callback.call(img);
     return; // 直接返回，不用再处理onload事件
    }
 img.onload = function () { //图片下载完毕时异步调用callback函数。
        callback.call(img);//将回调函数的this替换为Image对象
    };
};

jqury当中offset和position的区别

在jquery中position（）获取到的是元素距离最近带有定位的元素的距离，而offset（）的话获取的永远是距离浏览器文档的距离。

cursor：hand并不是标准的写法，很有可能没有效果，标准的写法应该是cursor：handler

如果一个父盒子没有设置高度，是由内容撑起的高度，后面的子盒子又想去继承父盒子的宽度和高度的话，是可以通过设置自身宽高来达到继承效果的
