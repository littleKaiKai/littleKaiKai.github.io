正则表达式概念

Created at 2014-10-11

正则表达式

  • 什么是正则表达式
    • 正则表达式是用来匹配的，包含普通字符或者是元字符（+ * ）的字符串对象
    • 在其他语言中，正则表达式就是一个字符串，在js中，正则表达式是一个对象
  • 创建正则表达式的方式
    • var r = /123/
    • var r = new RegExp(字符串或者字面量)
  • 正则表达式的作用：
    • 匹配 -> test
    • 提取 -> exec
      作用是用来匹配并且提取相应的内容，如果匹配到相应的内容了，就会把获取到的内容放到返回值中去了，返回值是一个数组或者null，数组中包含了：1. 所有被匹配到的项；2. index： 被匹配项在字符串中的所引号
      var r = /e/g 全局模式
      在正则中找分组的时候，只看左小括号出现的位置
      如果不想让分组的内容在结果中展示出来，只需要将（）替换为”（？：）“
      （）-> 分组并捕获
      （？：）-> 分组不捕获
    • 替换 -> replace
    • 正则表达式如果前后都由^ $的限制，那么其实还隐藏的含义是，中间的内容只能是我们规定好的的那两个字符之间的内容
